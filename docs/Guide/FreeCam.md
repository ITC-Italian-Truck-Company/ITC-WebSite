# FreeCam

# Primo step
- Recati sul percorso C:\Users\'nomecomputer'\Documenti\Euro Truck Simulator 2
- Trova config.cfg e aprilo con Blocco note o Visual Studio Code

# Secondo Step 
- Trova 'uset g_developer' e 'uset g_console', quindi cambia il valore da 0 a 1 (Ctrl+F essere utile per abilitare la ricerca rapida), salva ed esci.

# Terzo Step
- Ora puoi usare la Freecam nel gioco premendo "0" dalla barra numerica superiore,
puoi muoverti con il tastierino numerico.

# Note
- Utilizza Ctrl + F9 per teletrasportare il tuo camion dopo aver scelto la posizione con la freecam
(Ricorda di posizionare la visuale raso terra).
- Puoi impostare la velocità di movimento con 'uset g_flyspeed'

# Attenzione
- Dopo l'aggiornamento 1.22, viene automaticamente disabilitato Ctrl + F9 durante il lavoro Contratti esterni.
- Funziona anche su American Truck Simulator. L'unica differenza è il percorso del file config.cfg.
- Durante il teletrasporto, il rimorchio verrà montato automaticamente.
- Utilizzare g_console abilita anche la Console, tasto \ (accanto a !)
